## 📌 About this tool

This program converts markdown files into professional PDF documents. Markdown is a simple way to write text using basic formatting. You use symbols like hashes or asterisks to create headings, bold text, or lists. This program reads those files and saves them as PDF documents. It keeps your formatting intact and produces high-quality output for reading or printing. You do not need to understand code to use this tool. It runs directly on your Windows computer.

## 🛠️ Troubleshooting 🔧

If the program fails to open, verify your Windows version. Ensure you have the latest updates for your operating system. If the program closes unexpectedly, check your file path. Sometimes, files with very long names or complex characters in the folder name cause issues. Try moving your markdown file to a folder like "Documents" on your C: drive and try the conversion again.

If you encounter errors during the conversion, ensure no other programs are using the file you want to convert. Close the document in your text editor before you process it with md2pdfgo. This prevents file access conflicts.
